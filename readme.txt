This plugin picks up all your installed plugins in a list.
You can then choose which plugin you want to show your gratitude to.

== Description ==
This plugin is created for all who want to give credit to those developers who have created your favorite plugins.
The plugin brings up the name, url, version and the description of the selected plugin and puts it  in where you want it to appear.
Either on a page or in your widget, you might have a sidebar or in the footer.
You will show your gratitude for these developers by giving them a little credit by showing your visitors about the plugin that you are using on your website or blog.


== Installation ==
This section describes how to install the New Seed Credit plugin and get it working.
Use Add Plugin within the WordPress Admin panel to download and install this jNew Seed Credit plugin from the WordPress.org plugin repository (preferred method). Or download and unzip this plugin, then upload the /New Seed Credit/ directory to your WordPress web site\'s /wp-content/plugins/ directory
Activate the New Seed Credit plugin through the Installed Plugins Admin panel in WordPress.
